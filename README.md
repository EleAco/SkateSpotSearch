# SkateSpotSearch
全国のスケートボードが練習できるスポットを共有するSNSです。
投稿する画像の住所をGoogleMapで位置を取得できる。


# URL
簡単ログイン可能

# 使用技術
* Ruby:2.6.5 , Rails:6.0.2
* webpacker(bootstrap4)
* docker,docker-compose
* nginx,puma
* Rspec
未定　* Google map api
未定　*　terraform
未定　*　circle ci
